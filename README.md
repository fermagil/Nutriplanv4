## Descripción

Nutriplanv4 es un proyecto de banco de pruebas para el desarrollo de una aplicación de planeamiento nutricional. Actualmente se encuentra en fase de pruebas internas y no está listo para producción.

## Características Principales (en desarrollo)

✅ **Planificación Personalizada**: Crea planes nutricionales adaptados a tus objetivos  
✅ **Base de Datos de Alimentos**: Amplia colección de alimentos con información nutricional  
✅ **Seguimiento de Calorías**: Monitorea tu consumo diario de calorías y nutrientes  
✅ **Generador de Menús**: Crea menús balanceados automáticamente
